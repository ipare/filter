## 介绍

`@ipare/filter` 是 `Ipare` 的请求过滤器插件

请求过滤器可用于参数校验、身份验证、返回结果转换等

## 安装

```
npm install @ipare/filter
```

## 开始使用

请访问 <https://ipare.org>
